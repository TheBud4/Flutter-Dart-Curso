
# Dice Roller App

Um aplicativo Flutter simples desenvolvido como projeto de aprendizado no curso **Flutter & Dart - The Complete Guide [2025 Edition]**.

## Descrição

Esta aplicação implementa um rolador de dados D6 (6 faces). O projeto foi criado para praticar conceitos fundamentais do Flutter, como gerenciamento de estado, widgets e interação com o usuário.

## Objetivo

Consolidar conhecimentos em:
- Criação de interfaces com Flutter
- Manipulação de eventos
- Gerenciamento básico de estado
